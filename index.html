コマンド 説明 git branch -d {ブランチ名} 特定のブランチを削除する。 git branch -m {旧ブランチ名} {新ブランチ名} ブランチの名前を変更する。 git branch -r リモートリポジトリのブランチを確認する。 git branch -a 全てのブランチを確認する。 git switch {ブランチ名} 特定のブランチへ移動する。「git checkout」と同じ。 git checkout -b {ブランチ名} ブランチを作成し、作成したブランチに移動する。 git switch -c {ブランチ名} ブランチを作成し、作成したブランチに移動する。 git fetch リモートリポジトリの内容をローカルに持ってくる。 git remote リモートリポジトリの一覧を表示する。 git diff 作業ツリーのファイルの差分を表示する。 git reset {ファイル名} インデックスにある対象ファイルを作業ツリーに戻す。 （git addを取り消す） git stash 変更した内容を作業ツリーから一時的に退避させる。 git stash apply 退避させた内容を作業ツリーに戻す。
